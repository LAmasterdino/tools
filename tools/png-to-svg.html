PNG umwandeln, in ein Ziel-Repo speichern und direkt verlinken

Diese Version speichert deine GitHub-Zugangsdaten lokal im Browser, lädt die erzeugte SVG per GitHub Contents API in ein frei wählbares Repository hoch und zeigt dir danach den direkten Download-Link an.

1. Upload Lege Token, Owner, Repo, Branch und Zielpfad im Account-Menü fest.
2. Speichern Die SVG wird als Base64 über die GitHub Contents API erstellt oder aktualisiert.
3. Teilen Nach dem Upload bekommst du einen direkten Raw-Download-Link.
